Баг-репорт:

Функция (starts_with) принимает в качестве искомого символа всю строку целиком и возвращает True
  ОР: функция возвращает True, если строка начинается с заданного символа 
     (пример (starts_with("Training", "T") == True) искомый символ - "Т") 
  ФР: функция возвращает True, если ввести в качестве искомого символа всю строку целиком 
     (примере (starts_with("Training", "Training") == True) строка = искомый символ)
Серьезность: S2
Приоритет: Lowest 

Функция (end_with) принимает в качестве искомого символа всю строку целиком и возвращает True
  ОР: функция возвращает True, если строка заканчивается заданным символом
     (пример (end_with("Hello World", "d") == True) искомый символ - "d")
  ФР: функция возвращает True, если ввести в качестве искомого символа всю строку целиком 
     (пример (end_with("Hello World", "Hello World") == True) строка = искомый символ)
Серьезность: S2
Приоритет: Lowest
